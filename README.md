# 🌌 Zyvex Portfolio Website

A modern personal portfolio website built with Flask, HTML, CSS and JavaScript.

## ✨ Features

- 🌍 English / Persian language support
- 📱 Fully Responsive Design
- 🎨 Modern UI with Glassmorphism
- 📊 Animated Skill Bars
- 🍔 Mobile Navigation Menu
- ⚡ Smooth Animations
- 🌙 Dark Theme
- 🚀 Fast Loading

## 🛠️ Built With

- HTML
- CSS
- JavaScript
- Flask (Python)
